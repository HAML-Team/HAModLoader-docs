# Introduction to HAModloader
**HAModLoader* is a cross-platform, community-driven mod-loader for the mobile game Hybrid Animals. This mod-loader is based on an older version of the game (v185) with modifications to make the game have mod-loading capabilities and an API for modding.

## Downloading HAModLoader
### Android:
 * Download the latest released version of the apk file from the github repo
 * Open the folder you've downloaded it in
 * Install the apk
 * Open the game so it can set up the files.
 ### Information:
 To manage the mods and logs, open the file manager app and navigate to your `Android` > `obb` > `com.Trident35ro.HAModloader` (`/storage/emulated/0/Android/obb/com.Trident35ro.HAModLoader`) in there, to install a mod, locate the `mods` folder and put the dll in there. To troubleshoot an error, locate the `logs` folder and read the logs.
 ### Troubleshooting:
 * It says blocked by play protect, what do I do?
 Go into google play store, click on your profile or the 3 dots, click play protect, and turn off play protect, you may turn it back on after the installation finished

### Windows:
 * Download the latest released version for windows from the releases page
 * Go to the folder you've downloaded it in
 * Unzip the archive you've downloaded and put it in a place like Documents, so you don't accidentally delete it.
 * Run `HAModLoader.exe`
 * Let it initialize and you're ready to go.
 ## Information :
 * To manage mods and logs, open file explorer, navigate to the folder where you've stored the game folder in, and look for `mods` and `logs`
 * To Modify your save file, open the run prompn (Win + R) and put in `%localappdata%/Low/Trident35ro/HAModLoader`, it should open your save folder.
 ## Troubleshooting :
 * It says dangerous app blocked, is the game a virus?
 No, the game is not a virus. Go into Windows Defender and temporary disable your firewall or add an exclusion for the folder or file.
### MacOS:
 * Download the latest release for MacOS
 * Go to the folder you've downloaded the file to
 * Run HAModLoader.dmg
 * Drag the HAModloader Logo to the applications folder in the opened windows
 * Run the game and let it initialize
 ## Information:
 You can find the `Mods` and `Logs` folder at `/Users/<your-username>/Library/Application Support/HAModloader`. If you want to install a mod, locate the `Mods` folder and put the dll file in the folder. As of now, we have no information as to where the save files are stored.
### Linux (Any Distro):
 * Download the latest released version for Linux
 * Go to the folder you've downloaded it to.
 * Unzip the archive you've downloaded
 * Run HAModLoader.x86_64 (alternitavely, open the console at taht path and run `./HAModLoader.x86_64`)
 * Let the game initialize
 ## Information:
 In Linux, the `Mods` and `Logs` folders are located in ~/.local/share/HAModLoader. To install a mod,  put the .dll file in the `Mods` folder.
